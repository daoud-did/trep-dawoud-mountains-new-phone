## Common Issues & Solutions

### Problem: Menu toggle not working
**Solution:** Make sure `js/app.min.js` is loaded with `<script async>`

### Problem: Styles not applying
**Solution:** Verify `css/main.min.css` is linked in `<head>`

### Problem: Dark mode not saving
**Solution:** Check that localStorage is enabled in browser

### Problem: Icons not showing
**Solution:** Ensure Font Awesome CDN link is included and can be accessed

### Problem: Slow performance
**Solution:** 
1. Check image sizes
2. Run PageSpeed Insights
3. Enable server compression
4. Use CDN for assets
